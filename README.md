# sysDEC
projekt Android (Java + C++) do systemu *sysDEC*.  


## Uruchomienie

1. Otwórz projekt w **Android Studio**  
2. Zbuduj i uruchom na emulatorze lub urządzeniu

## Struktura

- `sys_dec_final` – główna aplikacja  
- `codetector`, `drawable`, `layout` – moduły i zasoby UI

## Wymagania

- Android Studio  
- Android SDK kompatybilny z projektem
